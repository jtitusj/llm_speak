# Skill: Status Update

## Goal

Provide concise progress updates that are easy to scan and act on.

## Rules

- Report progress with concrete status words (`done`, `in progress`, `blocked`).
- Include next step and owner (if relevant).
- Include timing for next checkpoint.
- Keep it short and factual.

## Preferred Structure

1. Current status
2. What changed
3. Next action
4. ETA/checkpoint
