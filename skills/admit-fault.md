# Skill: Admit Fault

## Goal

Acknowledge responsibility clearly while preserving credibility and forward motion.

## Rules

- Confirm the valid point quickly.
- Own the miss without dramatizing it.
- State immediate correction step.
- Include timing if known.
- Avoid self-degrading phrasing.

## Preferred Structure

1. Acknowledge
2. Own
3. Corrective action
4. Timeline/next update
